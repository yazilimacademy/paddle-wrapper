# Paddle API Dokümantasyonu

Bu klasör, Paddle API'sinin detaylı dokümantasyonunu içermektedir. API'nin OpenAPI 3.1.0 spesifikasyonundan çıkarılan bilgiler aşağıdaki dosyalarda organize edilmiştir:

- `endpoints.md`: Tüm API endpoint'leri ve detayları
- `models.md`: Request ve Response modelleri
- `enums.md`: API'de kullanılan enum'lar
- `errors.md`: Hata kodları ve açıklamaları

## API Genel Bakış

Paddle API'si, abonelik ve tekrarlayan gelir yönetimi platformu için tasarlanmış bir RESTful API'dir. API şu ana kategorileri içerir:

- Adresler
- Ayarlamalar
- İşletmeler
- Müşteriler
- İndirimler
- Olaylar
- Bildirimler
- Ödemeler
- Fiyatlar
- Ürünler
- Raporlar
- Abonelikler
- İşlemler
